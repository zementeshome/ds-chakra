# Theme

The `theme` tokens components is responsible to merge and exposes all the Design Token in one consistent object.

This _complete theme_ is then ready to be injected in the [`extendTheme`](https://chakra-ui.com/docs/theming/customize-theme#customizing-theme-tokens) theming function:

```jsx
import React from 'react';
import { ThemeProvider, extendTheme, theme } from '@loblaw/ds-chakra/theme';

// Extend default Chakra theme
const fullTheme = extendTheme(theme);

// Usage
<ThemeProvider theme={fullTheme}>
  {/***
    Your components tree
  */}
</ThemeProvider>;
```
